Here are the container technologies that I'm mostly working with currently.

Container Orchestrators

Kubernetes

Large scale container orchestration with self-healing and auto-scaling.

See More

Docker Swarm

Lightweight, easy-to-use, small scale container orchestration.

Container Runtimes

Docker Engine

The industry's de facto container runtime that runs on various Linux and Windows operating systems.

containerd

An industry-standard container runtime with an emphasis on simplicity, robustness and portability.

CRI-O

An implementation of the Kubernetes Container Runtime Interface to enable using Open Container Initiative compatible runtimes.
