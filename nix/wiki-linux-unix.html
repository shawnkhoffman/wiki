Welcome to my wiki on *nix operating systems.

I'm using these *nix-based platforms more than any other distros or families of operating systems.

Platforms

macOS

The UNIX-based OS that powers every Mac.

Ubuntu

A very popular distro based on Debian.

Red Hat

A powerful enterprise distro.

SUSE

Another popular enterprise distro.

Package Managers

APT

An intelligent package manager for Debian and Debian-based distros.

dpkg

The default package manager for Debian and Debian-based distros. Less intelligent than APT.

DNF

The next-gen version of the yum package manager. Also the default package manager since Fedora 22.

yum

An intelligent package manager for RPM-based distros.

RPM

The default package manager for RPM-based distros. Less intelligent than yum.

ZYpp

Powers Linux applications like YaST, Zypper and the implementation of PackageKit for SUSE.

pacman

The main package manager for Arch-based distros.

Snap

Allows developers to ship their applications packaged together with their dependencies.

Homebrew

An intelligent package manager for macOS that allows you to build and install software to your taste.

Shells

Bash

Bourne Again, this is the default login shell for most Linux distros.

Zsh

An extended Bourne shell with some Bash features, ksh, and tcsh.
